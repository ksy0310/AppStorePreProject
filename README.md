## :ledger: 프로젝트 내용
* 필수구현 - 검색화면
	- 검색어입력전 : 최근 검색어 목록
	- 검색어입력중 : 최근검색어일치항목리스트표시
	- 검색어 입력 후 결과화면 (앱아이콘, 타이틀, 별점, 스크린샷)
* 필수구현 - 상세화면
	- 앱아이콘
	- 타이틀
	- 스크린샷
	- ‘더보기’ 기능을 포함하는 앱설명 혹은 새로운 기능
* 테스트코드

* 개발환경
	- Xcode 14.2 (다른 버전 사용 금지)
	- Deployment target iOS13 이상
* 오픈소스
	- 사용가능
	- Cocoapod, Carthage 등 사용 시, 프로젝트에 프레임워크를 포함해주세요.
* 결과물
	- 프로젝트
	- 앱 실행 동영상 (참고용)
